def AnnounceHypotesis.announce_clbk (   req)

Parameters

This function retrieves the request field of the Announcement message. Inside the custom message is present the istances of classes PERSON, PLACE, WEAPON corresponding to the hypothesis to announce. The robot firstly reach the centre of the arena, then announce the hypothesis and finally returns to the initial location.
